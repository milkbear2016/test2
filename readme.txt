HyperSnap ver. 8

HyperSnap 8 is a screen capture and image editing
tool for Windows.

HyperSnap 8 is distributed on "try before you buy"
principle.  If used without a license beyond the
20 days free trial period, it will add Hyperionics
stamps to images that you capture or edit with it.
To purchase a license, visit our web site at 
http://www.hyperionics.com
